Introdução

Para que seja aplicada a técnica de priorização First Things Firts (FTF) não se pode haver requisitos com dependências. Para que essa prerrogativa seja atendida, é necessário se criar uma planilha de dependências entre os requisitos com a finalidade de filtrar somente os requisitos que se encontram sem nenhum vínculo.

Metodologia

Foram pegos os requitos da priorização de moscow e feita análise de cada relação com outro requisito, se o requisto possuir uma dependência é retirado.
